# Jni_extract
Author: **Jerem1ah**

_Auto-attach jni.bnta and pull all JNI types for Android shared libraries._

## Description:
When an Android JNI ELF shared library is opened, automatically attach a local jni.bnta type archive and pull all JNI-related types into the BinaryView. Also provides a manual command.
